## Statistical Analysis of Survival on the Titanic

### Overview

This project performs a full statistical analysis of the Titanic dataset, focusing on understanding which factors influenced survival.

Rather than treating it purely as a prediction problem, the goal was to examine the data from a statistical perspective and build a clear interpretation of the underlying relationships.

### What I did

- cleaned and prepared the dataset  
- explored distributions and missing data patterns  
- performed hypothesis testing on key variables  
- built regression models to quantify effects  
- interpreted coefficients and statistical significance  

### What makes this useful

The Titanic dataset is often used for simple classification tasks, but this project treats it as a full statistical study.

The emphasis is on interpretation, inference, and understanding the data rather than just maximizing prediction accuracy.

### What turned out interesting

Variables such as gender and passenger class had strong and consistent effects on survival.

Some variables appeared important initially but lost significance after controlling for others.

This highlights the importance of multivariate analysis.

### Why it matters

Statistical thinking is a core part of data science, especially when decisions depend on understanding relationships rather than just predictions.

This project reinforces that foundation and shows how to move from raw data to interpretable conclusions.
