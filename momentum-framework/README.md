## Momentum Strategy Framework

### Overview

This project builds a simple but reusable framework for testing momentum-based trading strategies.

The goal was to create a structure that can be extended and modified easily.

### What I did

- defined momentum-based signals  
- implemented entry and exit rules  
- tracked performance metrics such as returns and drawdowns  

### What makes this useful

Rather than focusing on one strategy, this project focuses on building a flexible structure that can support many variations.

This makes experimentation faster and more consistent.

### Why it matters

A good framework allows ideas to be tested efficiently, which is critical in both research and real-world applications.
