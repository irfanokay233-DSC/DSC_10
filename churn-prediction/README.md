## Customer Churn Prediction

### Overview

This project focuses on predicting customer churn using machine learning models.

The goal was not only to make predictions, but also to understand what drives customers to leave.

### What I did

- prepared and cleaned customer data  
- engineered features for modeling  
- trained and compared classification models  
- evaluated performance using standard metrics  

### What makes this useful

Churn prediction is a common real-world problem, but it is often treated as a pure modeling task.

This project emphasizes interpretation and decision-making, not just accuracy.

### Why it matters

Being able to identify at-risk customers allows businesses to act early and reduce losses, making this a high-impact application of data science.
