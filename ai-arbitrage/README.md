## AI-Based Arbitrage Detection

### Overview

This project explores how machine learning can be used to detect pricing differences across markets.

The goal is to identify opportunities while accounting for real-world constraints.

### What I did

- analyzed price data across multiple sources  
- built models to detect discrepancies  
- considered execution factors such as timing and costs  

### What makes this useful

This project connects machine learning with real-time decision-making, rather than static prediction.

It focuses on practical implementation rather than theoretical results.

### Why it matters

Bridging the gap between prediction and execution is one of the key challenges in applied data science.
