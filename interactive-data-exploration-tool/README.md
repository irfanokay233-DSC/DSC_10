# Interactive Data Exploration Tool

This project uses Jupyter Notebook to make data exploration more interactive and hands-on. Instead of presenting a fixed report, the notebook allows users to adjust inputs, run cells, change parameters, and immediately observe how the analysis responds.

The goal is to make data analysis more intuitive, engaging, and educational by helping users move from passive viewing to active discovery.

## Main Ideas

- Use Jupyter Notebook as an interactive analysis environment
- Allow users to modify inputs and immediately see changes in output
- Support exploratory data analysis through charts, summaries, and parameter changes
- Make data analysis easier to understand through active experimentation

## Tools

- Python
- Jupyter Notebook
- pandas
- matplotlib
- Data visualization
- Exploratory data analysis

## Purpose

The project is designed to bridge the gap between passive viewing and active discovery. It shows how notebooks can be used not only for analysis, but also as learning tools that help users understand patterns, relationships, and analytical choices in real time.
