## Macroeconomic Dashboard

### Overview

This project presents key economic indicators in a structured and visual way.

The goal was to make it easier to understand how variables like inflation, interest rates, and unemployment interact over time.

### What I did

- collected macroeconomic data from public sources  
- built time series visualizations  
- organized indicators into a dashboard-style layout  

### What makes this useful

Economic data is often scattered and difficult to interpret. This project brings multiple indicators together in a single view.

It allows patterns and relationships to become more visible.

### Why it matters

Clear visualization improves decision-making and helps translate complex data into actionable insights.
