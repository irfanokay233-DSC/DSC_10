About Me

Interactive Data Exploration Tool

This project uses Jupyter Notebook to make data exploration more interactive and hands-on. Instead of presenting a fixed report, the notebook allows users to adjust inputs, run cells, change parameters, and immediately observe how the analysis responds. The goal is to make data analysis more intuitive, engaging, and educational by helping users move from passive viewing to active discovery.

Data Wrangling Pipeline

Messy real-world data is rarely ready for analysis, so this project centers on building a clean and repeatable process for making data usable. It deals with missing values, inconsistent formatting, and duplicate records in a structured way. The main idea is to treat cleaning not as a one-time repairing job, but as a reliable pipeline that improves everything that follows.

Housing Price Analysis

Rather than jumping straight into prediction, this analysis begins with a simpler and more important question: what actually drives housing prices in the data? The project examines how features such as size, quality, and age relate to price, with an emphasis on interpretation and intuition. The goal is to identify relationships that are meaningful and stable, not just statistically convenient.

Statistical Analysis of Survival: Titanic Dataset

Using the Titanic dataset as a controlled case study, this project emphasizes statistical inference more than pure prediction. It applies hypothesis testing and regression techniques to examine which factors were most closely associated with survival. Because the setting is relatively simple and interpretable, it becomes a useful way to highlight core statistical reasoning in a transparent way.

Image Classification

This work compares a basic model with a convolutional neural network to show why image data generally requires more than standard machine learning methods. Since images contain spatial structure, the project focuses on how model design affects performance when that structure changes. It serves as a clear example of why deep learning became so important in computer vision.

Customer Churn Prediction

Even though a popular problem in predictive analytics, customer churn is not just a prediction problem but also an interpretation problem. This project focuses on identifying which customers are likely to leave while also examining the factors that drive that behavior. The goal is to balance predictive performance with interpretability so the results are useful for real business decisions rather than just model accuracy.

Momentum-Based Trading Framework

Momentum is one of the most intuitive ideas in quantitative investing/trading, which makes it a strong starting point for building a reusable testing framework. This project develops a simple structure for experimenting with momentum signals, execution rules, and performance evaluation. The emphasis is on clarity and flexibility, so the framework can be extended later into more serious strategy research.

Backtests vs Live Trading Performance

A strategy can look strong in backtesting and still fail in real trading. This project focuses on understanding why that gap appears by examining practical frictions such as transaction costs, slippage, and unrealistic execution assumptions. The goal is to build a more realistic evaluation framework and show how quickly apparent strong performance can weaken once ideal conditions are removed.

Crypto Market Inefficiencies

This promising project examines a potentially novel structural inefficiency in cryptocurrency markets that emerges under extreme leverage and volatility. The core idea is that the interaction of liquidation cascades, leveraged positioning, and short-horizon order flow can generate short-lived but repeatable pricing distortions that are not well explained by conventional market models. The project treats these events not as isolated anomalies, but as the visible outcome of an underlying mechanism that may be systematically identified and used for signal generation.

View Project

Capital Reallocation and Risk-Adjusted Performance

Position sizing is usually treated as secondary to the signal itself, but it can have a major effect on long-term outcomes. This project isolates the role of capital allocation, especially rules that increase exposure after losses. The analysis shows that while such rules can improve returns under some conditions, they can also produce sharper drawdowns and much greater tail risk. But the important conclusion is that it can substantially improve risk-adjusted performance when done successfully.
