## Quantitative Strategy Overview

### Overview

This project compares several common quantitative trading strategies under a consistent framework.

The goal is to understand how different approaches behave across similar conditions.

### What I did

- implemented multiple strategy types  
- evaluated performance using consistent metrics  
- compared results across strategies  

### What makes this useful

Instead of focusing on a single approach, this project provides a broader perspective on strategy design.

It helps highlight strengths and weaknesses across methods.

### Why it matters

Choosing the right strategy depends on context. This project helps build that intuition.
